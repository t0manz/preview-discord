# Previewer de Perfil do Discord

Ferramenta para pré-visualizar o perfil do Discord porque o Discord tem um rate limit que não me deixa experimentar quando quero mudar o meu perfil.

## Funcionalidades

- Cropper de avatar e banner (1:1 e 4:1 respetivamente <s>though n tenho a certeza do ratio do banner do disc mas whatever n me apetece mudar...</s>)
- Preview ao vivo do perfil, mensagens e call
- Cores no perfil para brincar
